## Features

- WCAG AA contrast validated — never ships unreadable text
- Semantic highlighting with fine-grained token colors
- Full editor UI theming (sidebar, tabs, status bar, terminal, git decorations)
- Designed for TypeScript, React, and JSX but works great with any language

## Also available for

Neovim, Kitty, Alacritty, WezTerm, Ghostty, iTerm2, foot, Windows Terminal, tmux, fzf, bat, delta, and lazygit.
